# Java RESTful Service (Line Interface Integration)

## Overview

This project is a production-ready RESTful API built with Java (Spring Boot) that integrates with LINE messaging platform for automated responses and user interaction.

## Features

* RESTful API with proper HTTP semantics
* Layered architecture (Controller, Service, Repository)
* Input validation and error handling
* Logging and monitoring ready
* LINE webhook integration

## Tech Stack

* Java 17
* Spring Boot
* MySQL / H2
* Maven

## Architecture

* Controller: Handles HTTP requests
* Service: Business logic
* Repository: Data access layer

## API Example

POST /api/messages
GET /api/users
